# 聪明钱 —— 两个视角

"聪明钱"指的是**两种不同的、相互独立的解读**。它们回答的是不同的问题；不要把它们混为一谈。

| 视角 | 工具 | 衡量什么 | 成本 |
|---|---|---|---|
| **情绪 / 持仓** | `ot smart` | Fear & Greed + 永续资金费率，作*反向*解读 | 免费、免密钥 |
| **链上巨鲸流向** | `ot whales` | 已打标签钱包的 ETH 余额 + Δ | 免费、免密钥 |
| *（可选）* **Antalpha** | MCP | 专业打标签的 巨鲸/VC/MM 钱包 + 美元信号 | **付费** |

## 1 · 情绪（`ot smart`）

CNN 股票 Fear & Greed（含 7 个子信号）、加密 Fear & Greed 指数，以及 BTC 永续资金费率。作**反向**解读：极度恐惧
= 已被洗盘出清 = 做多更安全；极度贪婪 = 过度拥挤 = 反向做空。

## 2 · 链上巨鲸流向（`ot whales`）

针对一个公共 Ethereum 节点的免密钥 JSON-RPC —— 无需密钥，无需账户。每次运行都会对余额做快照，并报告自上次以来的**变化量（delta）**：

- 币**流入**交易所钱包 → 潜在的**抛压**（利空）
- 币**流出**到冷存储 → **吸筹 / 托管**（利多）

它内置了**18 个公开打标签的钱包**，横跨五个类别（交易所、跨链桥、质押、基金会、巨鲸）。你可以在被 git 忽略的
`data/wallets.json` 中加入自己的钱包；它永不离开你的机器。

巨鲸流向是一个*背景*层，而不是一个触发器。在它改变你的仓位大小之前，先用资金费率和宏观门控来交叉验证它。

## 3 · 可选：Antalpha（付费）

如果你想要更丰富的、专业打标签的钱包，外加美元阈值信号，Antalpha 提供了一个**托管式 MCP** —— 一个 *skill/agent
级*的附加组件，它**并未**接入免密钥工具。它需要一次性注册（有它自己的 `antalpha-register` 工具），是**付费/计量**的，并且要求标注"Powered
by Antalpha AI"。

使用核心功能你永远不需要账户：`ot whales` 和仪表盘卡片始终保持免密钥。完整的、经过验证的设置 —— 端点、工具、钱包池、阈值 ——
都在仓库里：[**docs/SMART_MONEY.md**](https://github.com/orangejustin/OpenTrading/blob/main/docs/SMART_MONEY.md)。
