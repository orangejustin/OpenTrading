# Stage 3 — Quality: moat, cash flow, margin of safety

**Lineage:** Buffett Skills — https://github.com/agi-now/buffett-skills. Run on every long-list
survivor. Survivors only advance. A great supply-chain story with no moat or no margin of safety does
**not** pass.

## Moat tests (5 types — each needs a concrete test, not a label)

| Moat | The test |
|---|---|
| **Intangibles / brand** | Pricing-power test: *can it raise prices without losing customers?* Recognition ≠ moat. |
| **Low-cost advantage** | Source matters: scale (high), geographic/resource (very high, non-replicable), process/culture (moderate, imitable). Test: *could a rival match by spending $1B over 5 years?* |
| **Switching costs** | Signal: **net revenue retention > 120%**; deep data integration, retraining cost. |
| **Network effects** | Direct / indirect / data. Test: *if users left tomorrow, how much would they lose?* |
| **Efficient scale** | Market supports only 1–2 players profitably. |

- **Durability** — widening moat = rising gross margin / share gains / better renewals; narrowing =
  falling margin / share loss / "intensifying competition" in the filings.
- **Franchise vs commodity (Buffett 1991):** mentally raise price 10% — customers flee ⇒ commodity;
  they stay ⇒ franchise. Only franchises get an A here.
- **Goodwill read:** high goodwill **+ high ROE** = real economic goodwill; high goodwill **+ mediocre
  ROE** = overpaid (a yellow flag → carry to Stage 4 traps).

## Cash-flow / earnings metrics (the real engine)

- **Owner Earnings = Net Income + D&A − Maintenance CapEx − Required WC increases**
- **FCF = OCF − total CapEx**
- **ROIC = NOPAT / (Equity + Interest-bearing Debt − Cash)** — must exceed **WACC** to create value
- **Cash Conversion = OCF / Net Income**

## Quality thresholds (pass/investigate gates)

| Metric | Pass | Investigate |
|---|---|---|
| ROE (10-yr avg) | **> 15%** | < 15% |
| ROIC | **> 15%** (>20% exceptional) | < WACC = destroying value |
| Cash conversion | **> 90%** | **< 70%** → forensic check (Stage 4) |
| Debt / EBITDA | **< 2×** | > 2× |
| Interest coverage | **> 5×** | < 5× |

## Intrinsic value (3 methods) + margin of safety

- **Owner-earnings multiple (primary):** normalized 3–5yr owner earnings × multiple — **<8× for
  cyclicals → 20–25× for wide-moat compounders.**
- **Simplified DCF:** 10yr owner earnings + terminal; **discount rate = risk-free + 3–5%.**
- **Graham cross-check:** 7–10yr avg earnings × 10–15×.
- **Margin of safety:** default **buy at 60–70% of intrinsic value (30–40% discount).** Tier it:
  20–30% for predictable franchises → 40–50% for uncertain ones. **If intrinsic value can't be
  reliably estimated, don't invest — skip the name.**

## Region note

These metrics apply to **A-share / HK** too, but pull statements via the company's filings (no
FinancialJuice CN feed). For CN names, weight **cash conversion** and **related-party/goodwill** harder
— they're the most common manipulation vectors (handed to Stage 4). Value in ¥ / HK$, never USD.

**Output of Stage 3:** survivors with a moat verdict (franchise/commodity), the key quality ratios, an
intrinsic-value range, and whether today's price offers a margin of safety.
